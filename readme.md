# laravel-vue

## Installation

1 - Create database mysql "vue"
2 - cp .env.example .env;
3 - Configure .env params with database access.
4 - composer install;
5 - php artisan key:generate;
6 - php artisan migrate;
7 - php artisan passport:install;
8 - npm install;
9 - npm run dev;
10 - php artisan serve;
